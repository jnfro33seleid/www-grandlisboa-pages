# www-grandlisboa-pages

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存在，不依赖后端服务，可直接通过浏览器打开或部署到任意静态托管平台。

## 项目简介

此仓库主要存放历史或临时生成的 HTML 页面副本。每个页面均为独立完整的文档，包含自身的样式、脚本及结构。仓库不针对特定域名或网站，只作为页面归档的集中存储点。

## 目录结构

```
www-grandlisboa-pages/
├── pages/          # 存放各个 HTML 页面文件
├── assets/         # 页面引用的公共资源（如图片、样式表、脚本等）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立的页面归档。
- `assets/`：存放页面所需的静态资源，按需组织。

## 页面归档说明

- 所有页面均为纯静态 HTML，无服务器端逻辑。
- 页面之间相互独立，不共享运行时状态。
- 文件名或目录名通常反映页面内容或归档时间，但不代表任何官方或商业关联。
- 归档页面可能包含外部资源引用（如字体、CDN 库），这些资源在离线或网络受限环境下可能无法正常加载。

## 维护说明

- 欢迎通过 Issue 或 Pull Request 提交改进，但请注意保持页面独立性与完整性。
- 不建议将本仓库用于生产环境或作为主要发布渠道。
- 仓库不提供任何形式的担保或持续可用性承诺。

## 使用方式

1. 克隆或下载本仓库。
2. 使用浏览器直接打开 `pages/` 目录下的 HTML 文件。
3. 或部署至任意静态托管服务（如 GitHub Pages、Netlify 等）进行在线访问。
